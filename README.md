# 🤖🎤 AI Calculator with Voice Input

## 📌 Overview

AI Calculator is an intelligent web-based calculator that supports both **text and voice input** to perform calculations.

It not only handles standard arithmetic but also uses smart logic to interpret natural language expressions, making calculations faster and more interactive.

---

## 🎯 Features

* ➕ Basic arithmetic operations
* 🧮 Scientific calculations
* 🤖 AI-based expression understanding
* 🎤 Voice input for hands-free calculations
* ⚡ Real-time result display
* 🎨 Modern and responsive UI

---

## 🗣️ Voice Feature

* Users can speak expressions like:

  * "5 plus 3"
  * "square root of 144"
  * "10 percent of 500"
* Voice is converted to text using browser speech recognition
* The processed input is evaluated and result is displayed instantly

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Build Tool:** Vite
* **Voice Recognition:** Web Speech API
* **AI Logic:** Custom parsing / Lovable AI

## 🧠 How It Works

1. User inputs text or speaks a query
2. Voice input is converted to text
3. Expression is parsed using AI logic
4. Calculation is performed
5. Result is displayed instantly
